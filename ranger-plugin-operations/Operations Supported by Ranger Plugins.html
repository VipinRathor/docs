Operations Supported by Ranger Plugins
(As of Feb 2019; based on HDP 3.1 and HDF 3.3 line)
  1. HDFS
    - READ, WRITE, EXECUTE
  2. YARN
    - Submit-app
    - Admin-queue
  3. HIVE
    - CREATE, ALTER, DROP, INDEX, LOCK, SELECT, UPDATE, USE, READ, WRITE, ALL, REPLADMIN, SERVICEADMIN
  4. HBASE
    - CREATE, READ, WRITE, ADMIN
  5. ATLAS
    - CREATE, READ, UPDATE, DELETE
  6. KNOX
    - ALLOW
  7. KMS
    - CREATE, DELETE, ROLLOVER, GET, GET_KEYS, GET_METADATA, SET_KEY_MATERIAL, GENERATE_EEK, DECRYPT_EEK
  8. KAFKA
    - READ, WRITE, CREATE, DELETE, CONFIGURE, DESCRIBE, KAFKA_ADMIN
  9. SOLR
    - CREATE, UPDATE, QUERY, SOLR_ADMIN
  10. NIFI
    - READ, WRITE
  11. STORM
    - getTopologyInfo (common for these Storm operations: getTopologyPageInfo, getComponentPageInfo, setWorkerProfiler, getWorkerProfileActionExpiry, getComponentPendingProfileActions, startProfiling, stopProfiling, dumpProfile, dumpJstack, dumpHeap, setLogConfig, getLogConfig, debug)
